# subarray vs subsequence vs subset
- subarray - (consecutive)
    - maintains order and the continuity )
    - [1, 2, 3, 4] => [1, 2], [2, 3]
- Subsequence
    - maintains order by omitting the inbetween elements
- subsets
    - NO order or continuity
    - [1, 2, 3, 4] => [4, 1]
